InCHlib (Interactive Cluster Heatmap library) is an open source interactive Javascript library which provides an easy way to integrate, display and work with the hierarchically clustered data and with clustered heatmaps. It implements a raster-based HTML5 canvas visualization utilizing an open source Javascript framework KineticJS.
InCHlib is multiplatform and works on all common operating systems (Windows, Linux, iOS) in all major internet browsers (Internet Explorer, Mozzila Firefox, Google Chrome, Safari, Opera). Its speed is determined solely by the clientâ€™s hardware and internet browser.
