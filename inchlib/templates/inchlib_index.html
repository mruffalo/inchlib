InCHlib (Interactive Cluster Heatmap library) is an open source interactive visualization library developed in JavaScript. Because of its nature InCHlib is multiplatform and works on all common operating systems (Windows, Linux, iOS) in all major internet browsers (Internet Explorer, Mozzila Firefox, Google Chrome*, Safari, Opera).
InCHlib is raster-based visualization displayed through the HTML canvas technology. It was developed using KineticJS, JavaScript framework for HTML canvas interaction. Since InCHlib is a client-side application only the hardware and browser of choice determines its speed and smoothnes of graphic events.
