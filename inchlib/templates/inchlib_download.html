Dependencies
  • jQuery (tested with version 2.0.3)
  • KineticJS (tested with version 5.0.0)
Download inchlib
Installation
There is no need to install inchlib.js. As all JS libraries you just need to import the file in your code.
<script src="/path/to/inchlib.js"></script>
inchlib_clust.py
Dependencies
  • numpy (tested with version 1.7.1)
  • scipy (tested with version 0.12.0)
  • scikit-learn (tested with version 0.14)
  • hcluster (tested with version 0.2.0)
  • fastcluster (tested with version 1.1.8)
Download inchlib_clust
Installation on Linux
First you should install all python dependencies. Then you can proceed to install inchlib_clust using following commands:
sudo unzip inchlib_clust_0.1.0.zip cd inchlib_clust_0.1.0/ sudo python setup.py install python >>>import inchlib_clust
Installation on Windows
First you should install all python dependencies (and of course setuptools). Then you can proceed to inchlib_clust installation by running:
python setup.py install
