InCHlib
Download InCHlib
InCHlib requires the following dependencies to be installed:
  • jQuery (tested with version 2.0.3, download jQuery 2.0.3)
  • KineticJS (tested with version 5.0.0, download KineticJS 5.0.0)
Installation
To use InCHlib, copy inchlib.js into the selected folder, and import it from your code:
<script src="/path/to/inchlib.js"></script>
inchlib_clust
Download inchlib_clust
inchlib_clust requires the following dependencies to be installed:
  • numpy (tested with version 1.7.1)
  • scipy (tested with version 0.12.0)
  • scikit-learn (tested with version 0.14)
  • hcluster (tested with version 0.2.0)
  • fastcluster (tested with version 1.1.8)
Installation
On Linux, install inchlib_clust using following commands:
sudo unzip inchlib_clust-0.1.0.zip cd inchlib_clust-0.1.0/ sudo python setup.py install
On Windows, install inchlib_clust using following command:
python setup.py install
