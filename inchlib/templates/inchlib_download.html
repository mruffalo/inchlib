  • numpy (tested with version 1.7.1)

  • scikit-learn (tested with version 0.14)

Installation
On Linux, install inchlib_clust using following commands:
sudo unzip {{ files.inchlib_clust_file.name }} cd inchlib_clust-{{ files.inchlib_clust_file.version }}/ sudo python setup.py install
On Windows, install inchlib_clust using following command:
python setup.py install
